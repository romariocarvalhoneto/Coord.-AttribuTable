Coord. AttribuTable

Adds points' coordinates to attribute table, according to the chosen CRS.
This plugin makes easy to get the coordinates of many points into the attribute table,
even D°M'S" format, according to the chosen CRS.
It also supports plane coordinates. The coordinates are transformed to the chosen CRS,
if different from the original one.
